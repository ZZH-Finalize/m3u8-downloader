# m3u8 下载器 Edge 插件

这是一个 Edge 浏览器插件，用于管理 m3u8 视频下载任务。

## 功能特性

- **添加任务**: 提交 m3u8 下载链接到后端服务
- **删除任务**: 取消或移除下载任务
- **系统设置**: 配置服务器地址、端口、并发数等
- **任务列表**: 实时查看所有下载任务的进度和状态
- **自动刷新**: 可配置自动刷新任务列表

## 安装方法

### 1. 准备图标文件

插件需要以下图标文件（PNG 格式）：
- `icons/icon16.png` (16x16 像素)
- `icons/icon48.png` (48x48 像素)
- `icons/icon128.png` (128x128 像素)

您可以使用设计工具创建图标，或使用 `icons/icon.svg` 作为模板导出 PNG 文件。

### 2. 加载插件到 Edge

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 开启"开发人员模式"（左侧开关）
4. 点击"加载解压缩的扩展"
5. 选择 `extension` 文件夹

### 3. 使用插件

1. 确保后端服务已启动（默认 `http://127.0.0.1:5000`）
2. 点击浏览器工具栏中的插件图标
3. 点击"系统设置"配置服务器地址（如需要）
4. 点击"添加任务"提交下载任务
5. 在任务列表中查看下载进度

```
extension/
├── manifest.json      # 插件配置文件
├── popup.html         # 插件页面
├── popup.css          # 样式文件
├── popup.js           # 逻辑脚本
└── icons/
    ├── icon.svg       # 图标源文件
    ├── icon16.png     # 16x16 图标
    ├── icon48.png     # 48x48 图标
    └── icon128.png    # 128x128 图标
```

## API 配置

插件默认连接 `http://127.0.0.1:5000`，可在"系统设置"中修改：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 服务器地址 | 127.0.0.1 | 后端服务 IP |
| 端口 | 5000 | 后端服务端口 |
| 默认并发数 | 8 | 下载任务默认并发数 |
| 自动刷新间隔 | 2 秒 | 任务列表自动刷新间隔 |

## 后端服务

本插件需要配合 m3u8 下载器后端服务使用。启动后端服务：

```bash
python start_server_async.py --host 127.0.0.1 --port 5000
```

详细 API 文档请参考 [API.md](../API.md)。

## 开发说明

- 插件使用 Manifest V3 规范
- 使用 Chrome Storage API 保存配置
- 使用 Fetch API 与后端服务通信
